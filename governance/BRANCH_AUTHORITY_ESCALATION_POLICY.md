# Branch Authority Escalation Policy

Escalate when a change affects:

- source tiers;
- authority structure;
- doctrine/value ordering;
- ritual/legal practice;
- living religious representation;
- branch split/merge;
- Logos crosswalk;
- AI/no-human-in-loop permission claims;
- root authority artifact status.

Escalation path:

`contributor -> sub-builder -> branch reviewer -> branch head -> Noesis steward -> owner/arbiter`
