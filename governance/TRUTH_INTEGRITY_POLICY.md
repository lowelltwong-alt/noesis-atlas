# Truth Integrity Policy

## Core rules

1. Provenance is not truth.
2. Blockchain anchoring is not truth.
3. AI output is not primary evidence.
4. Web repetition is not independent confirmation.
5. Derivatives inherit lineage, not trust.
6. Every root artifact must declare truth mode and authority profile.
7. Branch truth is scoped to branch authority.

## Truth modes

- revelatory
- formal
- empirical
- legal_positive
- historical
- philosophical
- institutional_standard
- civic_normative
- synthetic_oracular_candidate

## Corrupted truth feedback controls

- detect circular source lineages;
- mark AI-generated sources;
- require source independence for promotion;
- quarantine unverified model outputs;
- downgrade claims if provenance is stripped or circular.
