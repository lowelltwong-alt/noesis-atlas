# Acceptable Use Policy

Noesis Atlas is for governed classification, study, comparison, auditability, and safe interoperability with Logos and other approved systems.

## Allowed by default

- private study;
- proposal drafting;
- branch-scoped classification;
- provenance modeling;
- Logos-compatible crosswalk drafting in proposal mode;
- audit and validation testing.

## Requires permission

- production deployment;
- public API hosting;
- commercial use;
- AI training or embeddings over the graph;
- religious/spiritual authority applications;
- automated governance decisions;
- exporting curated branch packages.

## Not allowed

- manipulation, coercion, surveillance, extremist propaganda, deceptive religious impersonation, or use as autonomous spiritual authority;
- using one branch to misrepresent another branch;
- stripping provenance and presenting derivatives as root authority;
- using AI-generated candidate claims as canonical truth.
